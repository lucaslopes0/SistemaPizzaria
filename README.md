# SistemaPizzaria
Sistema acadêmico para implementação de Design Patterns. 


## Como rodar o projeto (API Flask da Pizzaria)

### 1. Pré-requisitos

- Python 3.10+ instalado
- `pip` funcionando no terminal
- (Opcional, mas recomendado) VS Code para desenvolver

---
